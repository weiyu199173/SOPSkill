# 数字内容制作与营销推广 Skill

## 核心能力

### 1. 短视频制作
- 创意策划与脚本撰写
- 素材拍摄或 AI 生成
- 专业剪辑、配音、特效
- 平台适配优化（抖音、视频号、快手等）

### 2. 营销推广
- 平台算法深度理解与运用
- 精准投放策略与 A/B 测试
- 数据监控与效果优化
- ROI 提升 20%+

### 3. 直播运营
- 直播策划与准备
- 场控与互动管理
- 复盘分析与优化
- GMV 持续增长

## SOP 流程

### 短视频制作流程
1. 选题策划 → 热点追踪 + 用户洞察
2. 脚本撰写 → 3秒黄金开篇 + 完播设计
3. 素材制作 → 拍摄/AI 生成 + 剪辑
4. 特效包装 → 调色、字幕、音乐
5. 平台适配 → 格式、时长、标签
6. 发布推广 → 最佳时间 + 流量扶持

### 直播流程
1. 直播策划 → 主题、选品、流程
2. 前期预热 → 短视频预告、社群通知
3. 直播执行 → 开场、带货、互动、收尾
4. 数据复盘 → 观看时长、转化率、GMV
5. 后续跟进 → 粉丝维护、复购运营

## 核心指标

| 指标 | 目标值 |
|------|--------|
| 短视频完播率 | ≥ 40% |
| 互动率 | ≥ 8% |
| 转化率 | ≥ 5% |
| ROI | ≥ 1:3 |
| 直播平均观看时长 | ≥ 3分钟 |

## 爆款内容公式

### 短视频爆款公式
> 黄金3秒开场 + 价值/冲突/好奇 + 引导互动 + 行动号召

### 经典脚本模板
```
0-3秒：痛点/好奇点钩子
3-15秒：解决方案/内容展开
15-25秒：利益点/价值强化
25-30秒：引导关注/点赞/评论/转化
```

## 平台运营要点

### 抖音运营
- 发布时间：工作日 12-13点、18-22点
- 话题标签：3-5个精准标签
- 封面设计：大字标题 + 吸睛画面
- 互动策略：及时回复评论、引导转发

### 视频号运营
- 私域引流：社群、朋友圈预热
- 社交传播：设计分享诱因
- 内容调性：更偏向真实、情感、价值
- 直播联动：短视频 + 直播组合拳

## 营销方法论

### AARRR 漏斗模型
- 获客（Acquisition）：多渠道引流
- 激活（Activation）：首单体验优化
- 留存（Retention）：社群运营 + 内容触达
- 变现（Revenue）：多元变现方式
- 推荐（Referral）：裂变营销设计

### 数据驱动优化
- 每日数据监测
- A/B 测试迭代
- 爆款内容复用
- 用户画像精细化
