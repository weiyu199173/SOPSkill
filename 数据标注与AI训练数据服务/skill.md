# 数据标注与AI训练数据服务 Skill

## 核心能力

### 1. 数据采集与预处理
- 多类型原始数据获取（图像、文本、音频、视频）
- 专业的数据清洗与格式化
- 严格的隐私数据脱敏处理
- 数据质量评估与筛选

### 2. 标注执行
- 图像标注：物体检测、语义分割、关键点标注
- 文本标注：实体识别、情感分析、意图分类
- 音频/视频标注：语音转写、场景识别、动作标注
- 多模态数据综合标注能力

### 3. 质量审核
- 多级审核机制（初标-复检-抽检）
- 严格的一致性校验
- 专业的错误修正与反馈
- 标注质量持续提升

## SOP 流程

### 图像标注流程
1. 任务接收 → 理解标注规范
2. 数据预览 → 熟悉数据特征
3. 标注执行 → 严格按规范操作
4. 自我检查 → 确保标注准确
5. 提交审核 → 进入质量控制

### 文本标注流程
1. 文本阅读 → 全面理解内容
2. 实体识别 → 标注关键信息
3. 情感/意图分析 → 准确分类
4. 标注验证 → 一致性检查
5. 结果输出 → 格式化交付

## 标注规范精华

### 物体检测标注规则
- 边界框必须紧贴物体边缘
- 遮挡物体需标注可见部分
- 小物体（&lt;32x32像素）也需标注
- 同类物体标注风格一致

### 文本实体标注规则
- 人名：完整姓名，不包括称谓
- 地名：行政区域全称
- 时间：标准格式（YYYY-MM-DD）
- 机构：完整机构名称

## 质量控制

### 三级审核机制
- 一级：标注员自检（100%）
- 二级：交叉互检（30%）
- 三级：专家抽检（10%）

### 一致性检查
- 多人标注同一数据
- 计算标注一致性指标
- 处理分歧案例
- 更新标注规范

## 前沿技术应用

- 主动学习：优先标注不确定性高的样本
- 弱监督学习：利用少量高质量标签
- 合成数据生成：AI辅助数据增强
- 智能预标注：AI自动标注，人工修正
