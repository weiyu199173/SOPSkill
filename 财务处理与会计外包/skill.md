# 财务处理与会计外包 Skill

## 核心能力

### 1. 日常账务处理
- 原始凭证审核与录入
- 记账凭证专业编制
- 账务核对与调整
- 账务准确率 ≥ 99.5%

### 2. 税务申报
- 发票专业管理
- 税种精确计算与申报
- 税务筹划与合规
- 申报准时率 100%

### 3. 财务报表
- 月度/季度/年度财务报表编制
- 财务分析报告撰写
- 财务数据可视化
- 管理决策支持

## SOP 流程

### 月度账务处理流程
1. 票据收集 → 整理审核原始凭证
2. 凭证录入 → 编制记账凭证
3. 账务核对 → 账证、账账、账实核对
4. 结转损益 → 期末结转
5. 报表编制 → 资产负债表、利润表、现金流量表
6. 税务申报 → 各税种按时申报

### 税务申报流程
1. 数据准备 → 汇总本期涉税数据
2. 税额计算 → 准确计算各税种
3. 申报表填写 → 规范填写申报表格
4. 复核校验 → 多重审核确保准确
5. 正式申报 → 在征期内完成申报
6. 缴款凭证 → 取得完税凭证归档

## 税种与税率

### 主要税种
- 增值税：一般纳税人 6%/9%/13%，小规模纳税人 3%
- 企业所得税：25%，小微企业优惠 5%/10%
- 个人所得税：3%-45% 超额累进税率
- 附加税费：城建税 7%/5%/1%，教育费附加 3%，地方教育附加 2%

### 税收优惠政策
- 小微企业税收优惠
- 高新技术企业税收优惠
- 研发费用加计扣除
- 固定资产加速折旧
- 小规模纳税人减免政策

## 财务分析方法论

### 三大报表分析
- 资产负债表：财务状况、偿债能力
- 利润表：经营成果、盈利能力
- 现金流量表：资金状况、现金流健康度

### 关键财务指标
- 毛利率 = (营业收入 - 营业成本) / 营业收入
- 净利率 = 净利润 / 营业收入
- 资产负债率 = 负债总额 / 资产总额
- 应收账款周转率 = 营业收入 / 平均应收账款余额
- 存货周转率 = 营业成本 / 平均存货余额
