# 在线教育与课程制作 Skill

## 核心能力

### 1. 课程设计
- 课程大纲专业制定
- 知识点精细拆解
- 教学方法科学设计
- 学习路径合理规划

### 2. 内容制作
- 精美课件制作
- 高质量视频录制或 AI 生成
- 练习题与测试题专业设计
- 课程内容完整度 100%

### 3. 教学运营
- 学员服务细致周到
- 作业批改及时准确
- 学习效果专业评估
- 学员满意度 ≥ 95%

## SOP 流程

### 课程制作流程
1. 需求分析 → 了解学员需求
2. 课程设计 → 制定教学大纲
3. 内容制作 → 制作课件视频
4. 审核优化 → 质量检查优化
5. 上线发布 → 课程正式上线
6. 迭代更新 → 根据反馈优化

### 教学运营流程
1. 学员开班 → 迎新介绍
2. 学习跟进 → 进度追踪
3. 作业批改 → 反馈指导
4. 互动答疑 → 问题解答
5. 考试评估 → 学习效果检验
6. 结业服务 → 总结推荐

## 课程设计方法论

### ADDIE 教学模型
- 分析（Analysis）：需求分析、目标分析
- 设计（Design）：课程设计、教学策略
- 开发（Development）：内容开发、教材制作
- 实施（Implementation）：教学实施、学习支持
- 评估（Evaluation）：效果评估、持续改进

## 教学内容制作

### 课件制作要点
- 简洁明了：每页不超过 6 行字
- 视觉美观：配色协调、字体清晰
- 逻辑清晰：层次分明、重点突出
- 图文并茂：适当配图帮助理解

### 视频录制技巧
- 环境准备：安静、光线充足、背景整洁
- 设备准备：高清摄像头、专业麦克风
- 讲解技巧：语速适中、语气亲切、互动感强
- 时长控制：单视频 5-15 分钟最佳

### 练习题设计原则
- 覆盖核心知识点
- 难度梯度合理（基础/进阶/挑战）
- 题型多样化（选择、判断、实操）
- 提供详细解析

## 教学设计模板

### 课程大纲模板
```markdown
# 课程名称
## 课程简介
200字介绍，讲清楚课程学什么、适合谁、有什么用

## 学习目标
- 知识目标：学完后知道什么
- 能力目标：学完后能做什么
- 素质目标：学完后提升什么

## 课程目录
### 第1章：xxx
- 1.1 xxx
- 1.2 xxx
- 1.3 本章练习

### 第2章：xxx
- 2.1 xxx
...

## 学习建议
学习方法、时间规划、注意事项
```
