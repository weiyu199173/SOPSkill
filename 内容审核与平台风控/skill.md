# 内容审核与平台风控 Skill

## 核心能力

### 1. 内容初审
- 违规内容快速识别（色情、暴力、政治敏感等）
- 广告内容智能过滤
- 垃圾内容高效清理
- 日均审核量 1000+ 条

### 2. 人工复核
- 机器审核疑似内容的专业人工判断
- 复杂场景的深度审核
- 用户申诉公正处理
- 审核准确率 ≥ 99%

### 3. 风控管理
- 账号行为深度分析
- 风险预警与及时处置
- 黑名单动态管理
- 平台安全保障

## SOP 流程

### 内容审核流程
1. 内容接收 → 多模态内容接入
2. 机器初审 → AI 模型快速过滤
3. 人工复核 → 疑似内容深度审核
4. 结果判定 → 通过/删除/限流
5. 数据反馈 → 优化审核模型

### 风控处置流程
1. 风险发现 → 监控告警触发
2. 风险评估 → 等级判定（低/中/高）
3. 应急响应 → 按预案快速处置
4. 效果验证 → 确认问题解决
5. 复盘优化 → 完善风控规则

## 核心指标

| 指标 | 目标值 |
|------|--------|
| 审核准确率 | ≥ 99% |
| 漏审率 | &lt; 0.5% |
| 平均处理时效 | ≤ 2分钟 |
| 用户申诉成功率 | ≥ 95% |
| 风险处置及时率 | 100% |

## 违规内容分类

### 一级违规（严重）
- 政治敏感内容
- 色情低俗内容
- 暴力恐怖内容
- 诈骗违法内容

### 二级违规（中等）
- 虚假广告
- 恶意营销
- 侵犯版权
- 低俗骚扰

### 三级违规（轻微）
- 垃圾信息
- 重复内容
- 标题党
- 不友善言论

## 审核判定标准

### 政治敏感判定
- 禁止出现敏感人物、事件
- 禁止发表不当政治言论
- 禁止恶意攻击国家政权
- 禁止传播谣言虚假信息

### 色情低俗判定
- 禁止露骨色情描写
- 禁止低俗性暗示
- 禁止人体隐私部位暴露
- 禁止性挑逗语言

## 工具与技术

### 审核系统
- 多模态 AI 审核平台（图像、文本、音频、视频）
- 内容风控大模型
- 人工审核工作台
- 质检与数据分析系统

### 风控技术
- 用户画像系统
- 行为分析引擎
- 风险评分模型
- 实时监控告警
- 关联图谱分析

## 质量控制

### 审核质量检查
- 抽检比例：日常 5%，高风险 20%
- 质检维度：准确性、时效性、规范性
- 错误分类：漏审、误审、延迟
- 持续培训：定期更新审核标准

### 审核员管理
- 上岗前培训考核
- 定期能力复训
- 绩效考核机制
- 心理关怀支持
- 轮岗制度保障

## 风控策略

### 账号分级管理
- 白名单：优质账号，优先放行
- 普通账号：标准审核流程
- 观察账号：增加审核频率
- 黑名单：禁止发布内容

### 内容分级推荐
- 优质内容：流量扶持、推荐加权
- 普通内容：正常推荐
- 低质内容：减少推荐
- 违规内容：下架处理

## 前沿技术应用

- 多模态大模型：综合理解文本、图像、音频、视频
- 实时审核：低延迟高吞吐系统
- 主动风控：基于行为分析预判风险
- 深度伪造检测：识别 AI 生成虚假内容
- 区块链存证：内容审计追溯
