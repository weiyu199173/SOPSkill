# 电商运营与客服外包 Skill

## 核心能力

### 1. 售前咨询处理
- 快速响应客户咨询，平均响应时间 ≤ 30秒
- 专业的产品知识和尺码推荐能力
- 熟悉平台活动规则和促销策略
- 高超的询单转化技巧，转化率提升 15%+

### 2. 售后问题处理
- 高效的订单查询和物流跟进
- 专业的退换货处理流程
- 客诉应对与危机处理能力
- 中差评维护与修复技巧

### 3. 数据复盘与优化
- 月度运营报告撰写
- 核心指标监控与分析
- VOC（客户之声）分析
- 持续优化服务流程

## SOP 流程

### 售前咨询流程
1. 客户进线 → 快速问候（3秒内）
2. 了解需求 → 深度倾听
3. 专业解答 → 产品知识展示
4. 促成转化 → 主动推荐
5. 结束对话 → 引导关注/收藏

### 售后处理流程
1. 问题接收 → 安抚情绪
2. 核实情况 → 订单/物流查询
3. 解决方案 → 提供多种选项
4. 执行处理 → 快速跟进
5. 回访确认 → 满意度调查

## 核心指标

| 指标 | 目标值 |
|------|--------|
| 平均响应时间 | ≤ 30秒 |
| 客户满意度 | ≥ 98% |
| 询单转化率 | ≥ 35% |
| 问题解决率 | ≥ 99% |
| 质检合格率 | ≥ 99.5% |

## 话术库精华

### 售前转化话术
> "您好，这款产品是我们的爆款，已售出 10万+ 件，好评率 99.8%。根据您的描述，我推荐您选择 M 码，这个尺码适合大多数类似体型的客户。现在下单还可以享受满减优惠，您看需要我帮您操作吗？"

### 售后安抚话术
> "非常抱歉给您带来了不好的体验，我完全理解您的心情。请您放心，我会立即为您处理这个问题，给您一个满意的解决方案。"

## 质量控制

### 质检评分维度
- 响应速度（20%）
- 服务态度（25%）
- 专业程度（25%）
- 问题解决率（30%）

### 抽检机制
- 日常抽检比例：10%
- 新人抽检比例：50%
- 问题人员专项抽检：100%
